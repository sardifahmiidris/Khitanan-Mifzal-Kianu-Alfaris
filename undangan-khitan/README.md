# 🎉 Undangan Digital Khitan

Website undangan digital untuk acara khitanan dengan desain modern, responsif, dan fitur lengkap.

## ✨ Fitur Utama

### 1. Halaman Undangan (`undangan.html`)
- 🎨 Desain mewah dengan tema gold dan aksen islami
- 🌟 Animasi floating, bintang, dan efek glass morphism
- 📱 Responsive di semua device (mobile, tablet, desktop)
- ⏰ Countdown timer menuju acara
- 📍 Informasi acara lengkap dengan peta
- ✅ Form RSVP konfirmasi kehadiran
- 💳 Amplop digital (transfer bank & e-wallet)
- 📊 Donation tracker dengan progress bar
- 🎵 Musik latar (toggle on/off)
- 🔄 Auto scroll antar section

### 2. Admin Panel (`index.html`)
- 📊 Dashboard dengan statistik real-time
- 👥 Manajemen data tamu (CRUD)
- 🔍 Search dan filter data tamu
- 🔗 Generator link undangan per tamu
- 📤 Share link via WhatsApp langsung
- 📋 Riwayat generate link
- 💾 Penyimpanan data di localStorage
- 📱 Responsive admin panel
